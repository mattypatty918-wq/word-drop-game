# Word Drop Game

A fun and challenging word puzzle game. Drop letters to form words and score points!

## Features
- Hundreds of levels
- Daily challenges
- Leaderboards
- Hints system
- Beautiful graphics

## Build
```bash
npm install
npx cap sync
npx cap open android
npx cap open ios
```

## Tech Stack
- React + Vite
- Capacitor
- iOS & Android
